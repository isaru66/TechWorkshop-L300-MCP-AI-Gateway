# Exercise 01: Setup development environment

## Scenario

This is a hands-on lab and you are expected to install software on your device. There are two options for setting up your development environment: installing software directly on your local machine or using a GitHub Codespace. Choose the option that best fits your needs.

In this exercise, you will set up your development environment by forking and cloning the repository, installing the necessary software prerequisites, and verifying that everything is configured correctly for building MCP servers and configuring an AI Gateway.

## Objectives

After you complete this exercise, you will be able to:

* Fork and clone the workshop repository
* Set up a local development environment or GitHub Codespace
* Verify that all prerequisites are installed and configured correctly

## Duration

* **Estimated Time:** 10 minutes
